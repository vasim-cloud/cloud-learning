# Cloud Engineering - Day 5

## AWS Backup Project

### Objective
Learn and practice Linux basic commands on an AWS EC2 instance using EC2 Instance Connect.

## What I Practiced
- Started EC2 instance
- Connected using EC2 Instance Connect
- Practiced Linux commands
- Created folders and files
- Copied, renamed and deleted files
- Created a backup folder
- Stopped the EC2 instance to avoid unnecessary charges

## Commands Used

```bash
pwd
ls
mkdir
cd
touch
echo
cat
cp
mv
rm
clear
exit
```
